# FileOrganizer Pro

FileOrganizer Pro is a modern, dark-mode desktop utility that keeps your files clean and organized with a single click. It uses a hybrid smart-sorting engine, includes an undo stack for safety, and runs heavy work on a background thread so the UI stays responsive.

## Features

- **Smart Sorting** with master categories plus dynamic folder creation.
- **Undo** to revert the most recent operation instantly.
- **Dark Mode** interface powered by CustomTkinter.

## Installation

```bash
pip install -r requirements.txt
python main.py
```
